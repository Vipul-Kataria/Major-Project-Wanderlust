# 🌄 Wanderlust – Major Project

[![Node.js](https://img.shields.io/badge/Node.js-18.x-green)](https://nodejs.org/)  
[![License: MIT](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)  

Wanderlust is a travel & location-based web application built using Node.js. It enables users to discover destinations, view listings, and interact with travel content in a dynamic and user-friendly interface.

---

## 🌐 Features

- Discover travel destinations with rich details  
- View user-friendly listings and photos  
- Server-side built with Express and Node.js  
- Uses templating (EJS) for dynamic web pages  
- Organized structure for routes, models, middleware & views  
- REST APIs for data operations  
- Clean UI design with CSS  

---

## 🚀 Getting Started

### Prerequisites

- Node.js installed  
- npm (comes with Node.js)  

### Installation

```bash
git clone https://github.com/Vipul-Kataria/Major-Project-Wanderlust.git
cd Major-Project-Wanderlust
npm install

RUN The App
npm start

🗂 Project Structure
Major-Project-Wanderlust/
├── controllers/        # Logic for handling requests & responses
├── models/             # Data schemas and database interaction
├── public/             # Static files (images, CSS)

├── package.json        # Project metadata, scripts & dependencies
